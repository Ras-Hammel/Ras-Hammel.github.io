在vscode中配置java

发表于 2024-03-01 分类于 Java 阅读次数： 本文字数： 316 阅读时长 ≈ 1 分钟

在vscode里配置jdk。

jdk下载和配置

jdk下载网站：https://www.oracle.com/java/technologies/downloads/#jdk21-windows
注意要下windows版本，不要下错成其他版本了。。

下面是经典的系统变量配置：
1.添加系统变量JAVA_HOME: D:\Program\jdk\jdk-21
2.在系统变量Path中新建添加：D:\Program\jdk\jdk-21\bin

vscode配置

插件下载

插件商店搜索并下载Extension Pack for Java

设置

在vscode设置中搜索java.home，并将java.home的路径设置为jdk目录

然后就ok了，随便试一下吧。
